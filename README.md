# 用户浏览器诊断工具

可部署为ping.xxx.com站，供用户检测浏览器和网络信息。

### 检测结果包括：

#### 基础信息：

- 系统时间
- UA信息
- 系统信息
- 浏览器信息
- 语言信息
- JavaScript 状态
- Cookie 状态
- LocalStorage 状态

#### 网络信息：

- 本地IP、Local DNS
- CDN检测
- 国内热门网站
- 国外热门新闻站

#### 支持浏览器：

https://caniuse.com/es6-module

#### 支持多语言：

参照：[ISO-639](https://www.iso.org/iso-639-language-codes.html) 、
[ISO-3166](https://www.iso.org/iso-3166-country-codes.html)

- 简体中文
- 繁体中文
- 英语
- 日语
